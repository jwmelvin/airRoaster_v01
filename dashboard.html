Commissioning sequence
  1. Flash. Not a WebSocket command — run on the host with the board connected: ./verify.sh upload
  2. Sanity. Stay in manual; confirm sensors/OLED/Artisan behave. After a few minutes read control-cadence jitter (expect single-digit ms).
  3. Characterize. Set fan, set a moderate heat, let it settle, then step-test. Fan Heat Read the result; eyeball dT/Kp/tau/theta for sanity.
  4. Apply gains. Use the tight suggestion, or set the conservative set by hand if tight looks twitchy.  or  kp ki kd
  5. Calibrate feedforward. Still steady: set ambient, then auto-calibrate. Amb °C
  6. Close the loop. Set the inlet setpoint near the current inlet temp (bumpless), then nudge it and watch tracking. °C
  7. The real test. With the loop holding, deliberately change fan speed and confirm inlet temp barely moves — feedforward earning its keep. Tune from there. Fan
